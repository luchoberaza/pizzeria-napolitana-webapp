Instalación y ejecución de Pizzeria Napolitana WebApp

_______________________________________________________________________________________________
                									       |
1.**Instalar Node.js desde [https://nodejs.org](https://nodejs.org) (LTS) y reiniciar la PC    |
											       |
2.Descargar el repositorio (ZIP desde GitHub o git clone)				       |
											       |
3.Ejecutar CONSTRUIR.bat → hace todo solo (dependencias, DB, compilación)		       |
											       |
4.De ahí en adelante solo INICIAR\_PIZZERIA.bat para arrancar				       |
											       |
